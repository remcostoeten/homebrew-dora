# Dora

A lightweight Rust database manager for PostgreSQL, LibSQL, SQLite, and MySQL.

## Installation

### Homebrew (macOS/Linux)

```bash
# Add this tap
brew tap remcostoeten/dora

# Install
brew install remcostoeten/dora/dora
```

### macOS DMG

Download from [GitHub Releases](https://github.com/remcostoeten/dora/releases/latest)

### Linux

```bash
# AppImage
chmod +x Dora_*.AppImage
./Dora_*.AppImage

# or .deb/.rpm
sudo dpkg -i Dora_*.deb
# or
sudo rpm -i Dora_*.rpm
```

### Windows

Download the MSI or EXE installer from [GitHub Releases](https://github.com/remcostoeten/dora/releases/latest)
